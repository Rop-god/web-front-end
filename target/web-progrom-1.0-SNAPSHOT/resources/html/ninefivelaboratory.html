925移动开发实验室简介

移动开发实验室是针对我校计算机科学与技术、移动应用开发以及软件技术等专业设立的移动应用开发方向的专业实验室。主要设备有云桌面主机、教师云终端和50余台学生云终端，可根据需要通过服务器安装Android Studio、HBuilderX、Eclipse以及微信开发者工具等互联网及移动应用开发所需软件工具及开发环境，实验室占地面积108平方米，总价值约24.4万元。

实验室主要服务于《移动开发设计与实践》《移动应用开发》《Web程序设计与实践》 《微信小程序开发实战》等专业课程。 以移动应用人才培养为主线，打破以理论教学为中心的模式，不断完善和创新实践教学体系。根据学生的专业方向、层次水平、 兴趣爱好采用灵活的实践教学方式。除丰富的课程实践教学内容外，还合理地开设了Android应用开发、微信小程序开发、Web前端开发、 微信小程序开发实战等综合实训活动。此外，实验室将科研项目、企业项目、学科竞赛等移植到专业兴趣小组实践体系中，以项目驱动， 由浅入深，通过基础认知、综合应用、创新创业三个层面教学活动组成的阶梯式实践教学体系，全方位提升学生的综合应用能力。

实验室于2021年成功申报国家教育部第四批1+X移动应用开发证书试点。孕育了四川省应用型示范课程《移动开发设计与实践》及其课程团队。学生作品在各类国家级和省级学科竞赛中取得丰硕的成果。

实验室已开发多款不同平台的手机应用，包括iOS平台流媒体播放器，Android平台酷步软件，Windows Phone平台通讯中心，以及面向全平台的校园软件iXiyou和校园图书借阅查询系统客户端。

实验室配备了先进的计算设备和软件工具，为学生提供一个优越的学习和研究环境。我们鼓励学生在实验室中开展独立研究和项目实践，探索新技术的应用和解决方案的开发。此外，我们定期举办讲座、工作坊和技术竞赛，为学生提供展示和交流的平台，让他们能够与行业专家和同行分享他们的成果和见解。

作为一个开放和包容的学术社区，925移动开发实验室鼓励团队合作和思想交流。我们秉承着创新和协作的精神，致力于培养具备全球视野和竞争力的未来科技领袖。无论是追求学术研究的深度还是技术创新的广度，925移动开发实验室都是你探索和发展的理想之地。
